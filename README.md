# CppContactUtility

CppContactUtility is a simple C++ application for managing contacts. It provides basic functionalities such as adding contacts, displaying contact lists, saving contacts to a file, and loading contacts from a file.

## Features

- Add new contacts with names and phone numbers.
- Display the list of saved contacts.
- Save contacts to a file for future use.
- Load contacts from a file into the application.

## Getting Started

### Prerequisites

- Visual Studio

### How to Use

1. Clone the repository.

2. Open the project in Visual Studio.

3. Build and run the project using Visual Studio.
